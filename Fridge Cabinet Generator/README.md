# Fridge Cabinet Generator

Fusion 360 add-in prototype for a fridge cabinet stack layout editor.

Current scope:

- UI-only Fusion palette
- Cabinet, fridge, and clearance inputs
- Front-view vertical stack preview
- Click-to-add component palette
- Editable segment heights and order
- Calculated `z0`, `z1`, and height table

3D generation is intentionally not implemented yet.
